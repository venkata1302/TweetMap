# TweetMap
# Installation

### 1. Download
$ git clone https://github.com/venkata1302/TweetMap.git

$ cd TweetMap

### 2. virtualenv
$ python3 -m venv .

$ source bin/activate

### 3. Requirements
$ pip3 install -r requires.txt

### 4. Migrate
$ python3 manage.py migrate

### 5. Run
$ python3 manage.py runserver
## http://127.0.0.1:8000/home/
